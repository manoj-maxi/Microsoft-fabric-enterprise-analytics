# Microsoft-fabric-enterprise-analytics
End-to-end Microsoft Fabric analytics platform for telecom Revenue &amp; Cost Assurance. OneLake medallion lakehouse (Bronze→Silver→Gold) with PySpark notebooks, Dataflow Gen2 ingestion from Snowflake and REST APIs, a certified Direct Lake star-schema semantic model, advanced DAX with calculation groups, Purview governance (RLS/OLS, SOX/GDPR/Reg-W)
